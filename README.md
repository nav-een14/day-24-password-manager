# 🔐 MERN Password Manager

A simple Password Manager built using the MERN Stack. Store website credentials securely, search saved passwords, copy credentials, and manage them easily.

## 🚀 Features

- Add Password
- View Passwords
- Search Website
- Show / Hide Password
- Copy Username
- Copy Password
- Delete Password

## 🛠 Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Mongoose
- Axios

## 📂 Folder Structure

```
backend
│── models
│── routes
│── server.js
│── .env

frontend
│── src
│   ├── components
│   ├── App.jsx
│   ├── App.css
│   └── main.jsx
```

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside backend.

```env
PORT=5000
MONGO_URL=your_mongodb_connection_string
```
